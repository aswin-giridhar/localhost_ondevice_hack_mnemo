# Mnemo

A fully-offline general personal assistant that remembers across sessions.

On-device AI hackathon (2026-06-20). Runs entirely local: Ollama + Phi-4-mini, Cognee memory, Whisper voice, on-device vision. No cloud, no internet.

See design docs in the parent folder (`../docs/`, `../IDEATION.md`) — kept outside this repo intentionally.
